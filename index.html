Propriedades Position e Display no CSS

Propriedades position

A propriedade position, encontrada no CSS, define como um elemento pode ser posicionado (renderizado) no documento (página). Essa propriedade (position) pode ser acompanhada de outras, tais como, top, right, bottom, and left, que determinam como ficará a localização final do objeto, permitindo seu deslocamento, como será apresentado adiante.

Valores position:

Static

Este é o padrão. O elemento é posicionado de acordo com o fluxo normal do documento.

Exemplo

Relative

O elemento está posicionado em relação à sua posição normal no fluxo do documento.

Exemplo

Absolute

O elemento é posicionado em relação ao ancestral posicionado mais próximo.

Exemplo

Fixed

O elemento está posicionado em relação à janela de visualização.

Exemplo

Sticky

O elemento alterna entre uma posição relativa e fixa, dependendo da posição de rolagem.

Exemplo

Propriedades Display

Propriedade de display: A propriedade CSS ` display` define como um elemento aparece no fluxo do documento, afetando o layout, a visibilidade e a interação com outros elementos. Ela determina se os elementos se comportam como `block`, `inline`, `flex`, `grid` ou outros tipos especializados.

Valores:

Block

Exibe um elemento como um elemento de bloco (como (p)). Ele começa em uma nova linha e ocupa toda a largura.

Exemplo

Inline

O elemento permanece no fluxo do texto sem forçar uma quebra de linha. Ótimo para uso em elementos embutidos, como (span)ou (a).

Exemplo

Inline-block

Semelhante ao inline, mas permite modificações de largura e altura.

Exemplo

Flex

Converte um elemento em um contêiner flexível, permitindo o alinhamento eficiente de elementos filhos.

Exemplo

Table

faz com que um elemento HTML se comporte como uma tabela, mas sem precisar usar a tag table no HTML. Isso é útil para criar layouts que aproveitam o modelo de renderização de tabelas, como alinhamento vertical e distribuição automática de colunas.

Exemplo

Block flow

A propriedade display: block no CSS é utilizada para definir o comportamento de exibição de um elemento HTML na página. Quando um elemento é declarado como display: block, ele ocupa toda a largura disponível do seu contêiner e começa em uma nova linha, empurrando outros elementos para baixo.

Exemplo

Block flex

Elemento se comporta como bloco externamente e usa flex internamente. Na prática, funciona como um display: flex.

Exemplo

Block Grid

transforma o elemento em um container de grade (grid), permitindo organizar os filhos em linhas e colunas, pode ser usado como display: grid;

Exemplo

Block flow-root

cria um elemento de bloco que também inicia um novo contexto de layout independente, ele se comporta como um block, mas isola o conteúdo interno, evitando problemas como float escapando e margens se misturando.

Exemplo

Inline_flex

O valor Inline-flex faz com que o elemento se comporte em linha, porem utilizando o modelo de layout flexível para organizar seus elementos internos. Isso permite alinhar e distribuir itens de forma eficiente, mandentendo o elemento integrado ao fluxo de texto.

Exemplo

grid

O Valor grid transforma o elemento em um container de grade, permitindo a organização dos elementos internos em linhas e colunas esse modelo é amplamente usado para criar layouts mais completos e estruturados

Exemplo

inline-grid

O inline-grid possui funcionamento semelhante ao grid, porém com comportamento de elemento em linha. Ou seja, o elemento não ocupa toda a largura disponível, diferentemente do grid tradicional, mantendo-se alinhado com outros elementos inline.

Exemplo

flow-root

Por fim, o valor flow-root cria um novo contexto de formatação de bloco. Isso significa que o elemento passa a conter seus elementos flutuantes (floats) de forma independente, evitando problemas comuns de layout, como o colapso de altura.
